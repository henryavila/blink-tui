what you get
▦

cell-perfect layout

flexbox in the terminal. nest panes, blink aligns every glyph to the character grid — no off-by-one.

│

borders are glyphs

box-drawing characters, not CSS. single, rounded, or double lines — focus is a border recolour.

↵

keyboard-first

arrows, tab, hotkeys. focus is a ▶ caret or inverse fill. no mouse assumptions, no hover state.

nerd-font glyphs

a curated state + domain glyph set ships in. status reads as ✓ ✗ ◯ ◐ — never emoji, never SVG.

↻

calm by default

no transitions, no blur, no shadow. the only motion is the cursor blink and a braille spinner.

themeable

ships with Catppuccin Mocha. swap the palette token set and every component re-skins instantly.
